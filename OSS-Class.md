# 수업 내용 정리

### 1주차
**1. 오픈소스 소프트웨어 개요**
  - 소스 코드의 공개를 뜻하는 용어
  - 특징 : 전 세계의 개발자가 모여 오픈소스소프트웨어를 개발하려며 소스 코드 관리를 위한 도구와 원격 저장소 역할과 협업을 위한 서버가 필요

**2. 깃과 깃허브**
  - 깃 : 소스코드 관리를 위한 분사 버전관리시스템
  - 깃허브 : 깃 기반의 저장소 및 소프트웨어 협업 개발을 위한 웹 호스팅 서비스
